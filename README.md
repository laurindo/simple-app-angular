### Install Dependencies

```
npm install
```

### Run the Application

```
npm start
```

Now browse to the app at [`localhost:8000/index.html`][local-app-url].

## Directory Layout

```
app/                    --> all of the source files for the application
  styles/
    less/               --> pre-processor CSS
    main.css            --> default stylesheet generated by npm run build-css on package.json
    main.less           --> is the entry file to the lib less-watch-compiler read and generate the main.css above
  components/           --> all app specific modules
    header/                 --> component or directive header
      header-directive.js   --> directive for the specific model
      header.html           --> html for the specific module
      module.js             --> module for the specific component
  screens/              --> all the screens to required for the SPA application
    home/               --> screen home
      home_test.js      --> file for unit test
      home.ctrl.js      --> controller specific for home screen
      home.html         --> html to render home screen
      module.js         --> module to specific screen
      veiculos.json     --> simple json for mock data  
  config/
    routes.js           --> all routes of app
  factories/
    api-factory.js      --> file specific to format some data before to get from api-service
    module.js           --> module to specific factory
  services/
    http-service.js     --> service to manage all requests http
    module.js           --> module to specific factory
  constants/
    api-constant.js
  app.js                --> main application module
  index.html            --> app layout file (the main html template file of the app)
  index-async.html      --> just like index.html, but loads js files asynchronously
karma.conf.js         --> config file for running unit tests with Karma
e2e-tests/            --> end-to-end tests
  protractor-conf.js    --> Protractor config file
  scenarios.js          --> end-to-end scenarios to be run by Protractor
```

### Running Unit Tests

```
npm test
```
